A Practitioner's Reference Library

The architecture of debt recovery & restructuring in India

At the centre of this library stands the Insolvency and Bankruptcy Code — the 2016 reform that recast how India rescues and resolves a failing company, and the regime around which modern debt recovery is now organised. These volumes read the Code together with the laws that feed it: the contract and property doctrines beneath every security, the RBI framework that forces stress into the open, the SARFAESI self-help remedy, the tribunal that adjudicates, and the corporate code through which an entity is finally resolved — opening with a clause-by-clause guide to the Code’s first decadal overhaul, the 2026 Amendment.
